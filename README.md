# 智能短线操盘分析系统（网页版）

## 使用方式
双击 `index.html` 在浏览器中打开即可使用。

## 默认密码
`888888`（可在设置中修改）

## 功能
- **密码登录**：安全验证后进入主界面
- **自选股管理**：添加/删除自选股，自动保存到浏览器
- **实时行情**：通过新浪财经接口获取实时数据（部分场景使用模拟数据）
- **技术指标**：压力位/支撑位、MACD回踩线、量比/尾比
- **分时走势图**：Canvas 绘制的实时分时图
- **可视化图表**：量能对比柱状图、资金构成饼图、短线强弱评分
- **做T助手**：低吸/高抛区间计算、买卖记录、收益统计
- **系统设置**：密码修改、刷新频率、提醒设置

## 数据说明
- 优先通过新浪财经 API 获取实时行情
- 接口受限时自动降级为模拟数据
- 支持沪深两市所有 A 股
